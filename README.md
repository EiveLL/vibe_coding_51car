# 51 单片机智能小车课设

这是一个基于 STC89C52RC / 8051 的课程设计项目，目标是完成题目一的小车运行任务。

项目代码使用 Keil C51 编写，开发过程基本是边调硬件边改程序，全程由 Codex 辅助完成代码编写、调试整理和 Git 仓库维护。

## 当前功能

- TB6612 / L298N 类双路电机驱动控制
- 定时器 0 软件 PWM
- 10 ms 周期控制主循环
- 两路 ST188 红外巡线
- RC522 软件 SPI 初始化、寻卡、防冲突和 UID 读取
- 站点 UID 匹配
- 蜂鸣器站点提示
- 终点纯黑胶带停车

## 引脚分配

| 引脚 | 功能 |
| --- | --- |
| P1.0 | PWMA，左电机 PWM |
| P1.1 | IN1 |
| P1.2 | IN2 |
| P1.3 | IN3 |
| P1.4 | IN4 |
| P1.5 | PWMB，右电机 PWM |
| P1.7 | 蜂鸣器，低电平触发 |
| P0.0 | 左侧 ST188 OUT |
| P0.1 | 右侧 ST188 OUT |
| P2.0 | RC522 NSS / SDA |
| P2.1 | RC522 SCK |
| P2.2 | RC522 MOSI |
| P2.3 | RC522 MISO |
| P2.4 | RC522 RST |

P0 口没有内部上拉，如果 ST188 模块不是推挽输出，需要外接上拉电阻。

## 巡线逻辑

当前按两个 ST188 的输入状态控制小车：

| 左红外 | 右红外 | 动作 |
| --- | --- | --- |
| 0 | 0 | 直行 |
| 0 | 1 | 左修正 |
| 1 | 0 | 右修正 |
| 1 | 1 | 停车，并尝试判断是否为站点 |

输入需要连续稳定 3 次，也就是大约 30 ms，才会真正切换动作，用来减少传感器抖动。

当前速度参数：

- 直行速度：22%
- 转向时正向轮速度：40%
- 转向时反向轮速度：16%
- 上电起步补偿：约 0.2 s，45% PWM
- 站点停车后离站补偿：约 0.08 s，34% PWM

## 启动和停车

小车上电后前 0.5 秒只执行直行，不处理转向和终点停车。这样可以避免从黑线上启动时立刻误判为停车。

这 0.5 秒内仍然会读取 RC522，所以如果站点卡放在起点附近，程序也能识别到。

终点是一段纯黑胶带。两个红外都读到 1 且没有识别到登记卡时，小车进入终点停车状态，蜂鸣器响 1 秒，然后保持停车。

## RC522 站点识别

程序中已经登记了 4 张卡的 UID：

| 站点 | UID |
| --- | --- |
| 1 | 83 B1 00 07 |
| 2 | D3 97 BA E2 |
| 3 | 2C C0 31 6F |
| 4 | 83 F0 A1 05 |

运行时每 100 ms 轮询一次 RC522。读到登记 UID 后，小车停车 2 秒，蜂鸣器按站点号短响，例如 1 号站响 1 次，4 号站响 4 次。

同一张卡读取后会进入约 10 秒冷却时间，冷却期间不会重复触发同一张卡，避免一张卡被连续识别导致重复停车。不同站点卡不受这个冷却限制。

## 串口调试

程序会通过 UART0 打印读到的卡片信息。当前按 12 MHz 晶振配置 Timer2 波特率发生器，串口设置为 9600 bps、8 位数据位、无校验、1 位停止位。

输出格式示例：

```text
INIT OK
UID:83 B1 00 07 ST:1
UID:12 34 56 78 ST:unknown
```

系统初始化完成后会打印 `INIT OK` 或 `INIT ERR`。每次 RC522 成功读到 UID 时都会打印一行。未知卡不会触发站点停车，只会打印 `unknown`。

## 蜂鸣器

当前使用低电平触发蜂鸣器，接在 P1.7。

- 初始化完成：响 0.5 秒
- 普通终点停车：响 1 秒
- 站点停车：按站点号短响

## 工程文件

主要文件：

- `main.c`：主控制逻辑，包含定时器、PWM、电机、巡线、蜂鸣器和站点状态机
- `hardware.h`：引脚定义
- `rc522.c` / `rc522.h`：RC522 软件 SPI 和读卡接口
- `MOTOR_TEST.md`：调试记录和硬件说明
- `51car.uvproj`：Keil C51 工程文件

## 构建方式

使用 Keil uVision 打开 `51car.uvproj`，选择目标 `STC89C52RC` 后编译。

编译产物在 `Objects` 目录下，下载时使用生成的 `51car.hex`。

当前项目最近一次编译结果：

```text
0 Error(s), 0 Warning(s)
```
